# bibliomancy_machine
electronic bibliomancy machine

# What is bibliomancy?
bibliomancy is a form of divination performed by selecting a book - either randomly from some set of books or library or because you believe that book contains some truth - religious texts are common - and flipping to a random page, placing your finger on a random part of that page, and then reading from it to divine some truth. This is intended as a tool to perform this digitally, either by pointing it at a folder full of texts - pdf or raw text documents will eventually be supported - or pointing it at a single file and selecting randomly from that file. 

and yes you could just use fortune-mod for this. but then youd have to use fortune-mod's weird fucked up file formatting.
